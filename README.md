# ImpericaConverter
This is a Converter App made in Swift UI with new Observable macro

Features: 
1. Light and Dark Modes
2. Converts from Imperial system to Metric for these pairs:
- Weight
- Speed
- Distance
- Height
- Lenght
- Area
- Volume
- Temperature
